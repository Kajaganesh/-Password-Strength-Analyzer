# Password Strength Analyzer

## Description
This project analyzes password strength based on common security rules and generates a SHA-256 hash of the password. It demonstrates basic cybersecurity concepts such as strong password policies and secure password storage using hashing.

---

## Tools & Technologies
- Python 3
- hashlib
- Regular Expressions (re)

---

## How It Works
1. User enters a password
2. The script checks:
   - Password length
   - Uppercase and lowercase letters
   - Numbers
   - Special characters
3. The password is hashed using SHA-256
